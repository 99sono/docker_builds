# LaTeX CV Template

This is a simple LaTeX CV template.

## Files
- `main.tex`: The main LaTeX source file.
- `compile.sh`: Script to compile the document into a PDF.

## How to use
1. Edit `main.tex` with your information.
2. Run `./compile.sh` to generate `main.pdf`.

## Tips
You can use the `gemini` command to help you rewrite sections of your CV for better impact.
